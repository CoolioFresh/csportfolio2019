    • How the code Works

    There are many different types of code I know of, the ones listed on the right are the ones I know on the top of my head. The <p> </p> tags creates the paragraphs you're reading in this website, and the <h1> </h1> tags create the titles and headers of the website. Also, the <li> </li> and <ul> </ul> tags create the lists on the right side of the page, and the <img src="sample text"> adds the images of the website.

    • <h1> </h1> Adds Headers
    • <p> </p> Creates Paragraphs
    • <ul> </ul> Unordered Lists
    • <ol> </ol> Ordered Lists
    • <li> </li> Adds things to lists
    • <div> </div> Seperates lines of code
    • <br> </br> Breaks lines
    • <img src="sample text"> Adds images
