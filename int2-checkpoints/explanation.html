Explanation of the Code

Ellipses

Ellipses are a type of code that makes circles on the page. I used ellipses to make the beads on the rosary. The first
the first variable controls the x of the circle, or its horizontal location on the page and the y controls the vertical
location of it. The last two variables control the width and the height respectively. For example, in this code, the circle
is 30 pixels by 30 pixels, and is located at 80, 300.

Rects

Rects are a type of code that makes the rectangles on the page. I used rects to make the top and bottom parts of the
flag and the string of the rosary. The variables listed on the rects are very similar to ellipses, the first variable
controls the x of the circle, or its horizontal location on the page and the y controls the vertical
location of it. The last two variables of the code control the width and the height of the rectangle respectively.
For example, the rectangle is 350 pixels by 110 pixels, and is located at 0, 25.

Triangles

Triangles are a type of code that makes triangles on the page. I used triangles to make the stars on the flag. The variables
in the code translate to x1, y1, x2, y2, x3, y3. Since triangles have 3 points, the first two variables of the code determine the
first point of the triangles location. The next two variables of the code determine the second point of the triangles location,
x2, y2. Finally, the last two variables, x3, y4, determine the third point of the trianlges location.

Functions

Functions are special types of code that seperate code in Javascript. In my art piece, it is used in conjuction with Condtitionals
to change the color of the flag and spawn the rosary. It sections of lines of code in the code page to be used in different times,
like for example, activating a certain function after the mouse button is clicked.

Conditionals

Conditionals are special types of code that activates if certain criteria are met. In my art piece, it is used in conjuction with
functions to change the color of the flag. Conditionals consist of two main types of variables, if and else. "if" makes it so that if
one criteria is made, then the line of code under "if" will activate. One real world example of this is if the student is sleeping on a
weekday, then the student will get up early. "If" statements are similar to "else" statements, in that if the criteria is not met
then the code under "else" will activate. For example, if the student is not sleeping on a weekday, then the student will sleep in
for the day.

Loops

Loops are a type of code that plays a certain code over and over again if it is instructed to do so. In my art piece, it is used to
to make the beads on the rosary without having to code each one. Loops consist of the variables "for" and "i". "I" specifies the start point
of the shape, and tells the system to continously make more and more shapes.
